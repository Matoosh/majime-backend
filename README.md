# MajimeLIMS
MajimeLIMS is a Laboratory Information Management System created by a group of students.

## Usage
````
git clone https://github.com/Matoosh/majime-backend
````
#### In IntelliJ IDEA
```
File -> Open -> majime-backend/pom.xml
```
